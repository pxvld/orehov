Password Generator — это интуитивно понятный инструмент для создания надёжных паролей, который помогает пользователям защитить свои аккаунты от взлома.

Зачем это нужно?

Использование простых паролей (вроде qwerty123 или дат рождения) делает аккаунты уязвимыми для брутфорс-атак. Данный генератор создаёт сложные комбинации, которые невозможно подобрать или угадать.

Как это работает?

Вы просто выбираете параметры:

1. Длина пароля: от 8 до 32 символов.
2. Сложность: отмечаете галочками, какие символы использовать (цифры, знаки препинания, буквы разного регистра).
3. Нажимаете кнопку «Сгенерировать».
4. Программа гарантирует, что в пароле будет хотя бы по одному символу из каждого выбранного вами набора.
5. Готовый пароль можно сразу скопировать в буфер обмена кнопкой «Копировать».

Преимущества

· Локальная работа: Никаких серверов и интернета — пароли не покидают ваш компьютер.
· Гибкость: Вы сами решаете, какой должна быть сложность пароля.
· Простота: Минималистичный интерфейс, понятный любому пользователю.
· Кроссплатформенность: (Если на Python) Работает на Windows, macOS
