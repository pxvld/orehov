 Генератор паролей

Простое приложение для генерации случайных паролей с графическим интерфейсом.

 🎯 Цели

- Создать удобный инструмент для генерации надежных паролей
- Обеспечить безопасность пользователей в интернете
- Изучить основы разработки GUI-приложений на Python
- Получить готовый инструмент для повседневного использования

 📋 Задачи

- Создание базовой структуры проекта
- Разработка графического интерфейса
- Реализация алгоритма генерации паролей
- Добавление функции копирования в буфер обмена
- Обработка ошибок (проверка выбранных символов)
- Написание документации

 ⚙️ Функционал

- Генерация паролей длиной от 4 до 32 символов
- Выбор типов символов:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Спецсимволы (!@#$%^&*)
- Копирование пароля в буфер обмена
- Автогенерация первого пароля при запуске
- Проверка на пустой набор символов
- Уведомления об успешном копировании
Преимущества

· Локальная работа: Никаких серверов и интернета — пароли не покидают ваш компьютер.
· Гибкость: Вы сами решаете, какой должна быть сложность пароля.
· Простота: Минималистичный интерфейс, понятный любому пользователю.
· Кроссплатформенность: (Если на Python) Работает на Windows, macOS и Linux
